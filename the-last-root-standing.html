The Last Root Standing

Overview:

"The Last Root Standing" is a fun and comedic game developed as part of the Global Game Jam project at Unity Montreal offices. In this root-themed game, the player controls a root that must defend itself against an army of lumberjacks. The game was built using C#, Unity and WebGL

Contribution:

As part of the development team of seven people for this project, I was responsible for the following:

  • Developing enemy AI using a state machine
  • Implementing physics-based movement and collision detection for the enemies

Project Duration:

The project was completed over the course of the Global Game Jam event, which lasted 48 hours.
