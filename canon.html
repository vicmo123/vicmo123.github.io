Chicken Game

Overview:

The project is a basic first-person shooter game developed as a final project for my Graphics Libraries course. Set in a procedurally generated city, the player must defend itself against cactuses. The game was developed using C++, OpenGL, and a top-down/manager style architecture.

Contribution:

Working in a team of two people, I was responsible for the following:

  • Developing the first-person camera mechanics
  • Creating a UV mapping tool to speed up the process of adding texture to 3D shapes
  • Implementing a text generation tool using a sprite sheet
  • Designing game mechanics such as the State pattern, life system, and score system
  • The life system and the score system show renderer on screen using UI
  • Generating a procedurally generated city
  • Handling cube collision detection using Axis Aligned Bounding Box

Project Duration:

The project was completed over the course of one week.
