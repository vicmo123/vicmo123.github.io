Taco Truck Madness

Overview:

"Taco Truck Madness" is a VR game that immerses players in the chaotic world of a taco truck cook, where rats will try to ruin your day. The objective of the game is to serve tacos in time to keep the dinosaurs happy. The project was built using C# and Unity and Unity's VR Interaction Toolkit, and utilized agile/scrum methodology and time tracking software during development.

Contribution:

For this project we worked the whole class together in a team of six. Personally, I worked on the following:

  • Implementing AI for the rats using a state machine and nav mesh
  • Using Raycast to simulate fiel of view to locate and steal food from the player
  • Creating animations for the dinosaurs
  • Implementing animations using the animation state machine and blend trees for the rats and the dinosaurs
  • Implementing sounds using an event-based system
  • Working on the main Game Loop, which used a State Pattern
  • Creating a bound system and a respawnable component. Paired together objects can repsawn at a chosen location when out of bounds

Project Duration:

The project was completed over the course of one month, with weekly sprint cycles and daily team meetings.
