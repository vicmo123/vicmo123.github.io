Asteroid

Overview:

The Asteroid 2D game is a solo project that is inspired by the classic game Asteroids. The game was built using C# and Unity, and does not use any MonoBehaviours. It features a top-down/manager style architecture, and circle collision detection.

Contribution:

As the sole developer for this project, I was responsible for the following:

  • Implementing the game mechanics such as game flow, bullets system, score system, level system with increasing difficulty, and life system.
  • Implementing physics for flying and turning
  • Adding wrap-around functionality

Project Duration:

The project was completed over the course of a few days.
