Plinko

Overview:

"Plinko" is the first game I created using Unity. In this game, the player drops a ball down a peg board and tries to land it in a high-scoring slot at the bottom. The game features sound effects, a custom skybox, and trigger zones to detect when the ball lands in a scoring slot.

Contribution:

This was a solo project. This are some things I worked on:

  • Implementing physics-based movement for the ball using a rigid body
  • Designing and building the peg board layout
  • Creating trigger zones to detect scoring
  • Adding sound effects to the game
  • Implementing a custom skybox

Project Duration:

This project was completed in a few days as an introductory exercise to Unity game development.
