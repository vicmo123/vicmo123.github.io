Cowboy Shoutout Game

Overview:

Cowboy Shoutout is a solo project I created using Unity and C#. It's a game that challenges players to shoot targets as quickly as possible while shouting "Yeehaw!" before the time runs out. The game features a countdown timer, sounds, and a game reset function that allows players to play multiple rounds.

Contribution:

Working solo, I accomplished the following:

  • Designing the gameflow using a countdown timer and state pattern
  • Implementing sounds to enhance the player experience
  • Creating level design to make the game more engaging
  • Setting up the skybox to create a visually appealing game environment

Project Duration:

Cowboy Shoutout was completed over the course of several days and was the second Unity game I ever created.
